# Saber Downloader Pro 🚀

**برنامج متكامل لتحميل وتحويل وتشغيل الوسائط من الإنترنت**

## ✨ الميزات الرئيسية

- ⬇️ **تحميل الفيديوهات** من YouTube وجميع المواقع المدعومة (بدقة تصل إلى 4K)
- 🎵 **استخراج الصوت فقط** بصيغ MP3, M4A, AAC وغيرها
- 📋 **تحميل قوائم التشغيل** كاملة (YouTube Playlists, Channels)
- 🔄 **تحويل الملفات** بين جميع الصيغ (MP4, MP3, AVI, MKV, MOV, FLAC, WAV...)
- 🌐 **واجهة ثنائية اللغة** (العربية - الإنجليزية)
- 🌙 **الوضع المظلم** مدمج بالكامل
- 🎬 **مشغل وسائط مدمج** (يعمل عبر FFplay)
- 📝 **تنزيل الترجمات** ودمجها مع الفيديو
- 🖼️ **تحميل الصور المصغرة** بجودة عالية
- 🔍 **البحث المباشر** عن الفيديوهات داخل البرنامج
- 📊 **إحصائيات التنزيلات** وسجل كامل
- ⏰ **جدولة التنزيلات** لوقت محدد
- 📁 **منظم الملفات** تلقائياً حسب النوع
- 🌍 **ترجمة النصوص والترجمات** (Google Translate API)

## 📋 متطلبات التشغيل

- نظام التشغيل: **Windows 7/8/10/11** أو **macOS** أو **Linux**
- **اتصال بالإنترنت** للتحميل
- **ذاكرة عشوائية:** 512MB كحد أدنى

## 🛠️ الأدوات المضمنة

البرنامج يقوم بتحميل الأدوات التالية تلقائياً عند الحاجة:
- `yt-dlp` (أداة التحميل الأساسية)
- `FFmpeg` + `FFplay` (للمعالجة والتشغيل)

جميع الأدوات يتم تخزينها في مجلد `ffmpeg/bin` داخل البرنامج.

## 📥 كيفية الاستخدام

1. قم بفك ضغط الملف
2. شغّل `Saber Downloader Pro.exe`
3. في أول تشغيل، سيُطلب منك تحميل الأدوات (اضغط "نعم")
4. ابدأ بلصق رابط الفيديو والضغط على "جلب الجودات"
5. اختر الجودة المناسبة ثم "بدء التنزيل"

## 🔧 خيارات إضافية

- إلغاء التحميل بضغطة زر

# Saber Downloader Pro 🚀

**Complete Media Downloader, Converter & Player**

## ✨ Features

- ⬇️ **Download videos** from YouTube and 1000+ sites (up to 4K)
- 🎵 **Extract audio only** (MP3, M4A, AAC, etc.)
- 📋 **Download full playlists** (YouTube Playlists, Channels)
- 🔄 **Convert files** between all formats (MP4, MP3, AVI, MKV, MOV, FLAC, WAV...)
- 🌐 **Bilingual interface** (Arabic - English)
- 🌙 **Dark mode** fully integrated
- 🎬 **Built-in media player** (via FFplay)
- 📝 **Download subtitles** and embed in video
- 🖼️ **Download thumbnails** in high quality
- 🔍 **Direct search** for videos inside the app
- 📊 **Download statistics** and complete history
- ⏰ **Schedule downloads** for specific time
- 📁 **Auto file organizer** by type
- 🌍 **Text and subtitle translation** (Google Translate API)

## 📋 System Requirements

- **Windows 7/8/10/11** or **macOS** or **Linux**
- **Internet connection** for downloading
- **RAM:** 512MB minimum

## 🛠️ Built-in Tools

The app automatically downloads these tools when needed:
- `yt-dlp` (core downloader)
- `FFmpeg` + `FFplay` (processing & playback)

All tools are stored in `ffmpeg/bin` folder.

## 📥 How to Use

1. Extract the ZIP file
2. Run `Saber Downloader Pro.exe`
3. On first run, you'll be asked to download tools (click "Yes")
4. Paste video URL and click "Get Formats"
5. Choose quality then "Start Download"

## 🔧 Extra Options

- Change download folder
- Background downloading
- Cancel download anytime
- Play last downloaded file directly

No ads, no malware. Open source under MIT license.
